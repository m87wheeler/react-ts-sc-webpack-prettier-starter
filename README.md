# `react-ts-sc-webpack-starter`

A webpack starter built from the following packages:

-   ReactJS
-   TypeScript
-   Styled Components
-   Webpack
-   Babel
-   ESLint
-   Prettier


# To Run Prettier
On a specific file:
```
npx prettier --write <targetFile>
npx prettier --write src/App.tsx
```
On  all files:
```
npx prettier --write .
```
